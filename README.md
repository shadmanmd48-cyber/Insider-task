A simple and responsive e-commerce landing page built using HTML, Tailwind CSS, and custom CSS. This project focuses on creating a clean and modern UI with a smooth user experience.

Features:

Responsive navbar and hero section
Clean and minimal design
Feature highlight cards
Signup page UI
Mobile-friendly layout

Tech Stack:
HTML
Tailwind CSS
Custom CSS
Purpose

This project was built to practice frontend fundamentals and understand how to combine Tailwind with custom styling to create real-world UI layouts.
